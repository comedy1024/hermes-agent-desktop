🎯 快速开始

1. 配置 LLM 模型

首次使用需要在 WebUI 中配置 LLM API。打开 Hermes WebUI， 进入 Profiles 面板，添加你的 LLM Provider（如 OpenAI、Anthropic、DeepSeek、本地 Ollama 等）。

2. 常用终端命令

$ hermes # 启动交互式聊天 (TUI)
$ hermes gateway run # 手动启动 Gateway API
$ hermes --help # 查看所有命令
$ hermes skill list # 列出已安装的 Skills
$ hermes memory search "关键词" # 搜索记忆

3. 配置文件位置

文件路径说明

🔧 常用操作

添加 MCP 扩展

# 方式1：通过 WebUI → Profiles → MCP 配置
# 方式2：通过命令行
$ hermes mcp add my-server --command "npx my-mcp-server"

使用 Playwright 浏览器自动化

$ hermes
> 请用浏览器打开 https://example.com 并截图
Playwright + Chromium 已预装，Hermes Agent 可以直接控制浏览器进行网页操作。

WhatsApp Bridge

$ cd /opt/hermes/scripts/whatsapp-bridge
$ npm start
# 扫描二维码连接 WhatsApp

💾 数据持久化

⚠️ 容器重启后 /opt/data 以外的数据会丢失。请将重要文件保存在 /opt/data 目录下，或挂载 Docker Volume。
# Docker 运行时挂载数据卷
docker run -d -p 3000:3000 -p 3001:3001 -p 8787:8787 \
  -v hermes-data:/config \
